# Analysis Subproject

This subproject contains triplet-score and triplet-interpretation analysis scripts.

## Canonical scripts

- `triplet_reco.py`: triplet candidate reconstruction and plotting.
- `validate_triplet_interpretation.py`: branch interpretation validation with overlays.
- `diagnose_triplet_interpretation.py`: deterministic diagnostics for triplet interpretation.
- `plot_truth_triplet_0.py`: truth triplet plotting utility.
- `plot_truth_triplet_optimized.py`: optimized truth triplet plotting utility.
- `quick_truth_triplet.py`: quick truth triplet exploratory script.

## Notes

Run these scripts via their directory paths (for example,
`python analysis/triplet_reco.py ...`).
